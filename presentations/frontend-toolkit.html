Ja·va·Script

JS: User Experience

  • User Interaction
  • Event Handling

Car Analogy:

What actions can we perform on our car's objects?

What actions can we take on our web page's objects?

In JavaScript, these actions are called events. In our JavaScript, we can listen to these events and handle them accordingly.

Remedial jQuery (aka: $)

jQuery is a JavaScript library that allows you to use CSS selectors in your Javascript. (It also can do a lot more, but that's a whole other presentation).

jQuery is aliased as $, so you can use either one, but $ is shorter and easier to follow, so it's pretty rare to see just jQuery. 


Coin Toss: <span id="result"></span>
<button id="coin-toss-button">Toss Coin</button>

  $('#result').text(tossCoin());

  1. Create the ability to hide content when the user hovers over it.
<div class="js-hover">JS Hover.</div>

<script type="text/javascript">
  $('.js-hover').on('mouseover', function(){
    $(this).animate({opacity: 0}, 2000);
  }).on('mouseout', function(){
    $(this).animate({opacity: 1}, 2000);
  })
</script>
JS Hover.
<div class="hover">CSS Hover.</div>

<style type="text/css">
  .hover{
    transition-property: opacity;
    transition-duration: 2s;
  }

  .hover:hover{
    opacity: 0;
  }
</style>

  height: 0;
  opacity: 0;
  overflow: hidden;
  padding: 0;
  transform: translate(0, 100px);
  transition: all 200ms;
}

.tab-content.active{
  height: auto;
  opacity: 1;
  transform: translate(0, 0);
}
$('.js-tab').on('click', function(e){
  $('.tab-content').removeClass('active');

  var content = $(this).data('tab');

  $('#' + content).addClass('active');
});
